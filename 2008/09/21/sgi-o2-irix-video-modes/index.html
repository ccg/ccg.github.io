SGI O2 IRIX video modes

2008-09-21

Update 2018-02-10: I moved all the old files to a GitHub repo: https://github.com/ccg/irix-video-modes

This page has good information on custom video modes for the SGI O2: http://dart.ncsa.uiuc.edu/slevy/o2-stereo/

The author of that page has a zip file with useful files and information: o2-stereo.tar.gz

I wrote a couple video modes of my own:

  • 1400x1050 @ 60 Hz
    • 1400x1050_60.info
    • 1400x1050_60.vfo
    • 1400x1050_60.vfs
  • 1600x1024 @ 60 Hz
    • 1600x1024_60.info
    • 1600x1024_60.vfo
    • 1600x1024_60.vfs
    • 1600x1024_60.vfs.old

I also have a script to help me remember how to compile new modes: mkvfo.sh
